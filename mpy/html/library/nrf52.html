mpu6050 — functionality specific to the nRF52832 Bluefruit52¶

The mpu6050 module contains functions and classes specifically aimed at controlling nRF52832 Bluefruit52 modules.

Functions¶

mpu6050.MPU6050()¶

New a object MPU6050.

mpu6050.dmpInit()¶

mpu6050 init dmp mode.

mpu6050.setDMPEnabled(True)¶

set DMP Mode enable.

mpu6050.dmpGetFIFOPacketSize()¶

Return FIFO packet sizes.

mpu6050.getIntStatus()¶

Return interrupt status.

mpu6050.getFIFOCount()¶

Return FIFO counts.

mpu6050.getFIFOBytes()¶

Return FIFO bytes to buffers.

mpu6050.dmpGetQuaternion(fifoBuffer)¶

Calculation quaternion value and return to buffers.

mpu6050.dmpGetEuler(*fifoBuffer)¶

Calculation euler angle value and return to buffers(yaw,roll,pitch).

mpu6050.getTemp()¶

Return the soc temperature register value(°C).

mpu6050.getAccelX()¶

Return the Accel X register value.

mpu6050.getAccelY()¶

Return the Accel Y register value.

mpu6050.getAccelZ()¶

Return the Accel Z register value.

mpu6050.getGyroX()¶

Return the Gyro X register value.

mpu6050.getGyroY()¶

Return the Gyro Y register value.

mpu6050.getGyroZ()¶

Return the Gyro Z register value.
