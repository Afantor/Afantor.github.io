pca9685 — functionality specific to the nRF52832 Bluefruit52¶

The pca9685 module contains functions and classes specifically aimed at controlling nRF52832 Bluefruit52 modules.

Functions¶

pca9685.PCA9685()¶

New a object PCA9685 driver chip.

pca9685.reset()¶

reset PCA9685 register.

pca9685.freq(freq = 50)¶

set PCA9685 out pwm frequency.

pca9685.pwm(inedx, on, off)¶

set pwm out level.

pca9685.duty(index, value, invert)¶

set pwm out duty.

servo — functionality specific to the nRF52832 Bluefruit52¶

The servo module contains functions and classes specifically aimed at controlling nRF52832 Bluefruit52 modules.

Functions¶

servo.Servos()¶

New a object Servo by PCA9685.

servo.position(index, degrees)¶

set servo go to position(default:0~180).

servo.release(index)¶

release servo.
